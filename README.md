## ☁️ Firebase 部署 (Deployment to Firebase)

This project is configured for easy full-stack deployment to Firebase.

### 前置條件 (Prerequisites)

1. **Firebase 帳戶**: 確保您擁有 Firebase 帳戶，並已在 [Firebase Console](https://console.firebase.google.com/) 中創建新項目。
2. **Firebase CLI**: 如果尚未安裝，請全局安裝 Firebase CLI：
   ```bash
   npm install -g firebase-tools
   ```
3. **登錄**: 通過 CLI 登錄您的 Firebase 帳戶：
   ```bash
   firebase login
   ```

### 配置步驟 (Configuration)

1. 打開項目根目錄下的 `.firebaserc` 文件。
2. 將 `"your-firebase-project-id"` 替換為您實際的 Firebase 項目 ID。您可以在 Firebase 控制台的項目設置中找到此 ID。

### 後端設置 (Backend Setup)

本項目的後端 API 設計為在 Firebase Functions 上運行。在首次部署前，您需要安裝其依賴項。

1. 導航到 `functions` 目錄：`cd functions`
2. 安裝依賴項：`npm install`
3. 返回根目錄：`cd ..`

### 部署 (Deploy)

一旦配置完成，您可以使用提供的 npm 腳本部署整個應用程序（包括前端托管和後端函數）：

```bash
npm run deploy
```

此命令將：
1. 構建 React 應用的生產版本。
2. 將 `build` 目錄的內容部署到 Firebase Hosting。
3. 將 `functions` 目錄的後端 API 部署到 Firebase Functions。

要開始參與 JunAiKey 的開發，請按照以下步驟：

