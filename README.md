## Firebase 部署 (Deployment to Firebase)

This project is configured for easy full-stack deployment to Firebase.

### Prerequisites

1.  **Firebase Account**: Make sure you have a Firebase account and have created a new project in the [Firebase Console](https://console.firebase.google.com/).
2.  **Firebase CLI**: Install the Firebase CLI globally if you haven't already:
    ```bash
    npm install -g firebase-tools
    ```
3.  **Login**: Login to your Firebase account through the CLI:
    ```bash
    firebase login
    ```

### Configuration

1.  Open the `.firebaserc` file in the root of the project.
2.  Replace `"your-firebase-project-id"` with your actual Firebase project ID. You can find this in your project settings in the Firebase Console.

### Backend Setup (Firebase Functions)

The backend API for this project is designed to run on Firebase Functions. You need to install its dependencies once before the first deployment.

1.  Navigate to the `functions` directory: `cd functions`
2.  Install dependencies: `npm install`
3.  Return to the root directory: `cd ..`

### Deploy

Once configured, you can deploy the entire application (both frontend hosting and backend functions) using the provided npm script:

```bash
npm run deploy
```

This command will:
1.  Build the production version of the React application.
2.  Deploy the contents of the `build` directory to Firebase Hosting.
3.  Deploy the backend API from the `functions` directory to Firebase Functions.
